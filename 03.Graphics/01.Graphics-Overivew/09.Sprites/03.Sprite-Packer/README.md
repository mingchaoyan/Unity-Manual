# Sprite Packer

当设计sprite的时候，每个角色分开成单独的纹理文件是非常方便工作的。但是大量小片的纹理将会要求显存空间，这会导致运行时非常大的浪费。从性能角度来说，最好把分散的纹理打进一个图集。Unity提供了从单独的sprite到图集的工具。

Unity在场景后面处理图集纹理的生成和使用，所以用户不需要处理手动赋值。图集可以在Play模式打或者。

用户需要制定PackingTag，来标明需要打进图集的sprite
